## POST /api/search

Search for content within a dataset using grep or any arbitrary search command.

### Request body

<ParamField body="dataset" type="string" required>
  The dataset to search within.
</ParamField>

<ParamField body="flags" type="string" required>
  Flags to pass to the search command (e.g., `-r -n -i`).
</ParamField>

<ParamField body="folder_filter" type="string[]" required>
  Optional folders to filter the search (e.g., `["___private", "___user_auth"]`).
</ParamField>

<ParamField body="command" type="string">
  The search command to use. Defaults to `grep` if not specified. Allows using any arbitrary search command.
</ParamField>

### Example request

```json
{
  "dataset": "my-dataset",
  "flags": "-r -n -i",
  "folder_filter": ["___private"],
  "command": "grep"
}
```

### Response

Returns the search results from the executed command.
