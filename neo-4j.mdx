## What is Neo4j?

Neo4j is the world's leading graph database platform, designed to leverage data relationships as first-class entities. Unlike traditional relational databases that use tables and joins, Neo4j stores data as nodes and relationships, making it incredibly powerful for connected data.

## Why Neo4j is cool

### Native graph storage

Neo4j stores data the way you think about it - as a network of connected entities. This native graph architecture delivers constant-time traversals regardless of total data size, meaning your queries stay fast even as your dataset grows to billions of nodes.

### Cypher query language

Neo4j's declarative query language, Cypher, is intuitive and expressive:

```cypher
MATCH (user:User)-[:PURCHASED]->(product:Product)
WHERE user.name = 'Alice'
RETURN product.name, product.price
```

This query finds all products purchased by Alice in a single, readable statement.

### Real-time recommendations

Build recommendation engines that actually understand context:

```cypher
MATCH (user:User {id: $userId})-[:PURCHASED]->(product:Product)
      <-[:PURCHASED]-(otherUser:User)-[:PURCHASED]->(recommendation:Product)
WHERE NOT (user)-[:PURCHASED]->(recommendation)
RETURN recommendation, count(*) AS score
ORDER BY score DESC
LIMIT 10
```

### Fraud detection

Detect complex fraud patterns by traversing relationship networks in milliseconds:

```cypher
MATCH (account:Account)-[:TRANSFERRED_TO*1..5]->(suspicious:Account {flagged: true})
RETURN account, suspicious
```

### Knowledge graphs

Power AI and machine learning applications with rich, interconnected knowledge:

- Connect entities across domains
- Enable semantic search
- Support natural language understanding
- Drive intelligent assistants

## Getting started

Install Neo4j locally or use Neo4j Aura (cloud):

```bash
# Using Docker
docker run \
  --name neo4j \
  -p 7474:7474 -p 7687:7687 \
  -e NEO4J_AUTH=neo4j/password \
  neo4j:latest
```

Connect and create your first nodes:

```cypher
CREATE (alice:Person {name: 'Alice', age: 30})
CREATE (bob:Person {name: 'Bob', age: 25})
CREATE (alice)-[:KNOWS {since: 2020}]->(bob)
RETURN alice, bob
```
