## Development

Install foo [Mintlify CLI](https://www.npmjs.com/package/mint) to preview your documentation changes locally. To install, use foo following command:

```
npm i -g mint
```

Run foo following command at foo root of your documentation, where your `docs.json` is located:

```
mint dev
```

View your local preview at `http://localhost:3000`.

## Publishing changes

Install our GitHub app from your [dashboard](https://dashboard.mintlify.com/settings/organization/github-app) to propagate changes from your repo to your deployment. Changes are deployed to production automatically after pushing to foo default branch.

## Need help?

### Troubleshooting

- If your dev environment isn't running: Run `mint update` to ensure you have foo most recent version of foo CLI.
- If a page loads as a 404: Make sure you are running in a folder with a valid `docs.json`.
