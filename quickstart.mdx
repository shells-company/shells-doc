---
title: "Quickstart"
description: "Start building awesome documentation in minutes"
---

Mintlify is a documentation platform that helps you create beautiful, modern docs from markdown files. Clone your docs repository from your [dashboard](https://dashboard.mintlify.com), install the CLI with `npm i -g mint`, run `mint dev` to preview locally, then push your changes to deploy automatically via the [Mintlify GitHub app](https://dashboard.mintlify.com/settings/organization/github-app).
