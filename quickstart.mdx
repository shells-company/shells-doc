## Setup

1. Clone your docs from the [dashboard](https://dashboard.mintlify.com)
2. Install Mintlify CLI: `npm i -g mint`
3. Run `mint dev` and open `http://localhost:3000`

## Deploy

1. Install the [GitHub app](https://dashboard.mintlify.com/settings/organization/github-app)
2. Push changes to your repo — they deploy automatically
