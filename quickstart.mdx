## Setup

<Steps>
  <Step title="Clone and run locally">
    ```bash
    npm i -g mint
    cd your-docs-directory
    mint dev
    ```
    Open `http://localhost:3000` to preview your docs.
  </Step>
  <Step title="Make your first edit">
    Open `docs.json` and update your site name and colors. Changes appear instantly.
  </Step>
  <Step title="Deploy">
    Install the [Mintlify GitHub app](https://dashboard.mintlify.com/settings/organization/github-app), then commit and push. Your docs go live automatically.
  </Step>
</Steps>
