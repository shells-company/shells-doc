## 1. Set up locally

```bash
npm i -g mint
mint dev
```

Open `http://localhost:3000` to preview your docs.

## 2. Make your first edit

Open `docs.json` and update your site name and colors:

```json
{
  "name": "Your Project",
  "colors": { "primary": "#0D9373" }
}
```

## 3. Deploy

1. Install the [GitHub app](https://dashboard.mintlify.com/settings/organization/github-app)
2. Push your changes
3. Your docs go live automatically
