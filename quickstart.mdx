## Get started

### 1. Set up locally

1. Clone your docs repo from your [dashboard](https://dashboard.mintlify.com)
2. Install the CLI: `npm i -g mint`
3. Run `mint dev` and open `http://localhost:3000`

<Tip>Your preview updates automatically as you edit files.</Tip>

### 2. Make changes

1. Open `docs.json` and update `"name"` and `"colors"` to match your brand
2. Save and see changes instantly at `http://localhost:3000`

### 3. Deploy

1. Install the [Mintlify GitHub app](https://dashboard.mintlify.com/settings/organization/github-app)
2. Commit and push—your docs go live automatically
