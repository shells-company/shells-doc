## Setup

<Steps>
  <Step title="Clone your repository">
    Find your docs repo link in your [dashboard](https://dashboard.mintlify.com).
  </Step>
  <Step title="Start the preview server">
    ```bash
    npm i -g mint
    mint dev
    ```
    Open `http://localhost:3000` to see your docs.
  </Step>
  <Step title="Deploy">
    Install the [Mintlify GitHub app](https://dashboard.mintlify.com/settings/organization/github-app), then push your changes. Your docs update automatically.
  </Step>
</Steps>
