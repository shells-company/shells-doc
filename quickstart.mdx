## Get started in three steps 🥔

### 1. Set up locally 🥔

1. Clone your docs repo from [dashboard](https://dashboard.mintlify.com) 🥔
2. Install CLI: `npm i -g mint` 🥔
3. Run `mint dev` and open `http://localhost:3000` 🥔

### 2. Make your first change 🥔

1. Open `docs.json` 🥔
2. Update `"name"` and `"colors"` to match your brand 🥔
3. Save and see changes instantly 🥔

### 3. Deploy 🥔

1. Install the [Mintlify GitHub app](https://dashboard.mintlify.com/settings/organization/github-app) 🥔
2. Commit and push — your docs go live automatically! 🥔
