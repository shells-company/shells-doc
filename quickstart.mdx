## Prerequisites

- A Mintlify account with a [GitHub repository](https://dashboard.mintlify.com) for your docs
- Node.js installed on your machine

## Install the CLI

```bash
npm i -g mint
```

## Run your docs locally

1. Clone your docs repository
2. Navigate to the docs directory
3. Start the dev server:

```bash
mint dev
```

Your docs are now live at `http://localhost:3000`. Changes auto-refresh as you edit.

## Make your first edit

Open `docs.json` and update your site name and colors:

```json
{
  "name": "Your Project Name",
  "colors": {
    "primary": "#0D9373"
  }
}
```

Save the file and see your changes instantly.

## Deploy

1. Install the [Mintlify GitHub app](https://dashboard.mintlify.com/settings/organization/github-app)
2. Commit and push your changes
3. Your docs go live automatically
