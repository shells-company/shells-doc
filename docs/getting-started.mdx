Welcome to the getting started guide. This page will walk you through the basics of setting up and using the documentation.

## Prerequisites

Before you begin, make sure you have the following:

- A Mintlify account
- Node.js installed on your machine
- A text editor of your choice

## Quick Setup

1. Clone the repository
2. Install dependencies
3. Run the development server

```bash
npm install
npm run dev
```

## Next Steps

Once you have the development server running, explore the rest of the documentation to learn about customization, writing content, and deploying your site.
