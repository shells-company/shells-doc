## Update v2025.12.5 (mainly affecting Changelogs) — 29/12/2025 @ 11:00pm UTC.

Hello all,

It has been a little while since we last shared an update, and this one is a smaller housekeeping update focused primarily on how changelogs and updates are structured going forward.

### What is changing?

- We have introduced a new versioning system for updates and changelogs which follows this specifc format: `vYEAR.UPDATE.PATCH`.
- Small fixes and adjustments will now be reflected by patch numbers, e.g. `.1`, `.2`, `.3`, etc, instead of separate numbered updates.

### Why does this matter?

- This change is intended to make updates easier to understand across the year.
- This change makes changelogs clearly show when an update for any service or experience on Roblox is released.
- Any related fixes that are made will be grouped under the same main update.
- This change overall reduces any confusion and keeps update history more consistent.

No functionality has been changed as part of this update. This is purely a structural improvement.

We understand that this is **another small update**, especially ahead of a larger update planned before 2026, but we still appreciate your **continued support** towards us at ilike's Productions!
