## Ricky's Phone Call Radio

**Requirements:**

- Vapi for phone number / orchestration
- Phone call comes in → Press 1 for radio, Press 2 to choose a song, Press 3 to queue a song
- Play music through Spotify Connect
- Just keep playing until a button is pressed

**Flow:**

```text
Press 2 → "What song do you want to listen to?"
  songRef = GET USER INPUT()  // waits for user voice
  → "I got [X Y Z song] — 1 to confirm, 2 to deny?"
  → return songRef
  → Play songRef as if a radio (loop back to top)

Press 3 → "What song do you want to queue?"
  songRef = GET USER INPUT()  // waits for user voice
  → "I got [X Y Z song] — 1 to confirm, 2 to deny?"
  → return songRef
  → Add songRef to Spotify queue
  → Loop back to top
```
