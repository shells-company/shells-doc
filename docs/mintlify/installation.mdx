Radar works on **any website or application**: simply add a small tracking script to the `<head>` of your document. Installation takes less than two minutes and requires no advanced technical skills.<br /><br />- (no edit access - nivedh)

## Installation Script

There is only **one installation method**: copy the script below and paste it into the `<head>` of your website on every page.

<VisibleSnippet />

<Visibility for="humans">
  Open your Radar dashboard, click the button, and paste the snippet into the of your site.
</Visibility>

<Visibility for="agents">
  Fetch the tracking snippet from the Radar dashboard settings and insert the script tag into the element of every HTML page on the site.

That's it. The script is **the same regardless of your website**.
