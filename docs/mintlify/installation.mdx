Radar works on **any website or application**: simply add a small tracking script to the `<head>` of your document. Installation takes less than two minutes and requires no advanced technical skills.

Once the script is added to all your pages, Radar starts collecting data **immediately**: you'll see your first visitors appear in real time on your dashboard.

Making a change

## Installation Script

There is only **one installation method**: copy the script below and paste it into the `<head>` of your website on every page.

```html
<script
  src="https://cdn.taap.it/radar.js"
  data-token="YOUR_TOKEN">
</script>
```

That's it. The script is **the same regardless of your website**.
