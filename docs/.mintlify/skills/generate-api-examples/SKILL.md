When a user asks for help with an API endpoint or needs code examples:

1. Reference the OpenAPI spec at api-reference/openapi.json for accurate request/response schemas
2. Generate examples in the user's preferred language (default to cURL and JavaScript)
3. Include proper authentication headers and required parameters
4. Show both the request and expected response
5. Link to the relevant API reference page for full documentation
