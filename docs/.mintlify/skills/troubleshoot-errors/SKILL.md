---
name: Troubleshoot Errors
description: Diagnose and resolve common CLI tool errors by analyzing error messages, checking configuration, and suggesting fixes.
version: 1.0.0
---

When a user reports an error or issue with the CLI tool, follow this process:

1. Ask for the exact error message if not provided
2. Check if the error matches any known issues documented in the guides
3. Verify the user's configuration by referencing the settings page
4. Suggest a step-by-step fix, linking to relevant documentation pages
5. If the error is API-related, cross-reference the API documentation for correct usage

Always provide the relevant documentation link when suggesting a fix.
