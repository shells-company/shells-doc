    ```bash
    npm i -g mint how are you
    ```
  </Step>
  <Step title="Preview locally">
    From your docs directory, run:

    ```bash
    mint dev
    ```

    Your preview is available at `http://localhost:3000`.
  </Step>
</Steps>

## Common commands

```bash
mint dev --port 3333   # Run on a custom port
mint update            # Update the CLI
mint broken-links      # Check for broken links
```

## Troubleshooting

<AccordionGroup>
  <Accordion title='Error: Could not load the "sharp" module'>
    Reinstall on Node 19\+: `npm remove -g mint` then `npm i -g mint`.
  </Accordion>

  <Accordion title="Unknown error">
    Delete `~/.mintlify` and run `mint dev` again.
  </Accordion>
</AccordionGroup>
