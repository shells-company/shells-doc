## المصادقة

تتم مصادقة جميع نقاط نهاية واجهة برمجة التطبيقات باستخدام رموز Bearer ويتم اختيارها من ملف المواصفات.

```json
"security": [
  {
    "bearerAuth": []
  }
]
```
