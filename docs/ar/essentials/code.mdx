## الكود المضمّن

للإشارة إلى `كلمة` أو `عبارة` على أنها كود، ضعها بين علامتَي التنصيص العكسي (\`).

```text
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```

## كتل الأكواد

استخدم [كتل الأكواد المسيّجة](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) بإحاطة الكود بثلاث علامات تنصيص عكسية، واتبع علامات التنصيص الأولى باسم لغة البرمجة الخاصة بالمقتطف للحصول على تلوين بنية الكود. اختياريًا، يمكنك أيضًا كتابة اسم الكود بعد لغة البرمجة.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

````md
```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
````
