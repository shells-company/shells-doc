## الصورة

### باستخدام Markdown

تتيح لك [صيغة markdown](https://www.markdownguide.org/basic-syntax/#images) إضافة الصور باستخدام الكود التالي

```md
![title](/path/image.jpg)
```

لاحظ أن حجم ملف الصورة يجب أن يكون أقل من 5 ميغابايت. وإلا، نوصي بالاستضافة على خدمة مثل [Cloudinary](https://cloudinary.com/) أو [S3](https://aws.amazon.com/s3/). ثم يمكنك استخدام هذا الرابط وتضمينه.

### باستخدام التضمينات

للحصول على مزيد من التخصيص للصور، يمكنك أيضًا استخدام [التضمينات](/writing-content/embed) لإضافة الصور

```html
<img height="200" src="/path/image.jpg" />
```

</Tip>

### iFrames

يقوم بتحميل صفحة HTML أخرى داخل المستند. يُستخدم غالبًا لتضمين مقاطع الفيديو.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
