---
title: "إضافة كتلة صفحة المجموعة"
---

<Warning>
  لا تنطبق هذه الصفحة على عمليات التثبيت التي تمت في عام 2025 أو ما بعده لتطبيقات Depict Shopify
</Warning>

كتلة Depict Collection Page هي الكتلة التي تعرض صفحة المجموعة. سنوضح لك أدناه كيفية إضافة الكتلة إلى قالب المجموعة الافتراضي.

يمكنك أيضاً إضافتها إلى أي قالب مجموعة آخر لديك وكذلك إلى أي صفحات أخرى ترغب في عرض مجموعة عليها - فقط تأكد من تكوين معرّف مجموعة (collection handle) في إعدادات الكتلة إذا كانت الكتلة مستخدمة على صفحة ليست صفحة مجموعة.

<Steps>
  <Step title="الخطوة الأولى">
    اختر قالب المجموعة في محرر السمة.

    <Frame caption="انقر للتكبير">
      <img
        src="https://cdn.statically.io/gh/depict-org/documentation-images/main/shopify/selecting-collection-template2.svg"
        alt="فتح محرر قالب صفحة البحث"
        style={{aspectRatio: 3866/2075, width: "100%"}}
        ref={element => {
              // In prod, they have some zoom thing which breaks aspectRatio, work around that. Unfortunately this is not in the SSR'd output so there's still some CLS.
              const direct_parent = element?.parentElement;
              if(direct_parent?.matches?.("[data-rmiz-content]")) {
                  Object.assign(direct_parent.style, {
                      width: "100%",
                      display: "flex",
                      justifyContent: "center",
                      alignItems: "center"
                  })
              }
              const parents_parent = direct_parent?.parentElement;
              if(parents_parent?.matches?.("[aria-owns^='rmiz-modal']")) {
                  parents_parent.style.width = "100%";
              }
          }}
      />
    </Frame>
  </Step>
  <Step title="الخطوة الثانية">
    أضف كتلة Depict Collection Page. قد ترغب أيضاً في إخفاء كتلة المجموعة القديمة في هذه الخطوة.

    <Frame caption="إضافة كتلة صفحة البحث إلى قالب البحث" style={{ maxWidth:"500px" }}>
      <img
        src="https://cdn.statically.io/gh/depict-org/documentation-images/main/shopify/adding-collection-block2.svg"
        style={{aspectRatio: 1280/2104, width: "100%",}}
        ref={element => {
          // In prod, they have some zoom thing which breaks aspectRatio, work around that. Unfortunately this is not in the SSR'd output so there's still some CLS.
          const direct_parent = element?.parentElement;
          if(direct_parent?.matches?.("[data-rmiz-content]")) {
              Object.assign(direct_parent.style, {
                  width: "100%",
                  display: "flex",
                  justifyContent: "center",
                  alignItems: "center"
              })
          }
          const parents_parent = direct_parent?.parentElement;
          if(parents_parent?.matches?.("[aria-owns^='rmiz-modal']")) {
              parents_parent.style.width = "100%";
          }
      }}
      />
    </Frame>
  </Step>
  <Step title="تم">
    غيّر الخيارات القابلة للتكوين كما تريد واضغط على حفظ

    <Frame caption="تمت إضافة كتلة صفحة المجموعة">
      <img
        src="https://cdn.statically.io/gh/depict-org/documentation-images/main/shopify/collection-page-added2.svg"
        style={{aspectRatio: 3866/2702, width: "100%"}}
        ref={element => {
          // In prod, they have some zoom thing which breaks aspectRatio, work around that. Unfortunately this is not in the SSR'd output so there's still some CLS.
          const direct_parent = element?.parentElement;
          if(direct_parent?.matches?.("[data-rmiz-content]")) {
              Object.assign(direct_parent.style, {
                  width: "100%",
                  display: "flex",
                  justifyContent: "center",
                  alignItems: "center"
              })
          }
          const parents_parent = direct_parent?.parentElement;
          if(parents_parent?.matches?.("[aria-owns^='rmiz-modal']")) {
              parents_parent.style.width = "100%";
          }
      }}
      />
    </Frame>
  </Step>
</Steps>

<Tip>
  الآن بعد أن أضفت صفحات قوائم منتجات Depict إلى موقعك، نوصي بإزالة جميع آثار (بما في ذلك JavaScript للفرز والتصفية) كتل المجموعة القديمة، لتحسين سرعة الصفحة.
</Tip>
